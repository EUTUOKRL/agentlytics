agentlytics is a desktop app that helps you track activity and performance of AI coding assistants like Cursor, Windsurf, Claude Code, VS Code Copilot, and others. It offers a clean, easy-to-use dashboard to see your coding agent stats all in one place.

---

## 🖥️ What is agentlytics?

agentlytics presents a clear view of how different AI coding tools are helping you during your work. Instead of checking each tool separately, agentlytics gathers analytics into one dashboard you can open on your Windows machine. This makes it easier to manage and improve your AI-assisted coding experience.

---

## ⚙️ Features

- Collects data from popular AI coding tools including Cursor, Windsurf, Claude Code, VS Code Copilot, Zed, Antigravity, OpenCode, and Command Code.
- Shows detailed usage stats such as keystrokes, suggestions accepted, time saved, and active hours.
- User-friendly charts and tables for quick understanding.
- Works locally on your Windows machine, keeping your information private.
- Supports quick filtering by date, tool, and coding projects.
- Lightweight and runs smoothly without slowing down your computer.
- Automatically updates to include new AI coding assistants as they become available.

## 📋 How to Use agentlytics

Once agentlytics opens, it automatically connects with supported AI coding assistants running on your computer. Here’s what you can do:

- **View Dashboard:** See an overview of how your AI coding agents performed today or over any period you select.
- **Filter Data:** Use the filters to narrow down by agent, project, or date.
- **Check Detailed Reports:** Click on any AI tool to see more detailed stats like acceptance rate and active coding time.
- **Export Reports:** Export your data as CSV to review or share with others.
- **Settings:** Adjust preferences such as which AI assistants to track or enable automatic startup with Windows.

## 🔧 Troubleshooting Common Issues

- **App Won’t Open:** Restart your computer and try again. Make sure your Windows is up to date.
- **No Data Showing:** Ensure your AI coding assistants are running before opening agentlytics. Data starts collecting when those tools are active.
- **Installer Won’t Run:** Confirm you downloaded the Windows installer file, not a different package. Try running as administrator (right-click the file, select “Run as administrator”).
- **Dashboard Shows Incorrect Data:** Close agentlytics and reopen it. Your data will refresh automatically.
